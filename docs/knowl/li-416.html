Item 26.2.b.

We can use induction on \(n \geq 2\) to prove that if (i) and (ii) hold, then there is a tree with the given ordered degree sequence. The statement is true for \(n=2\text{,}\) and we assume it is true for all such sequences of length \(n-1\text{.}\) Consider a sequence \(d_1 \geq d_2 \geq \cdots \geq d_n\) satisfying (i) and (ii). Find the last \(d_i > 1\text{.}\) Remove \(d_n\) from the sequence and replace \(d_i\) with \(d_i-1\text{.}\) Explain why (i) and (ii) hold for the resulting sequence of length \(n-1\text{.}\) Use this fact to create a tree with degree sequence \(d_1 \geq d_2 \geq \cdots \geq d_n\text{.}\)
