For any \(a \in \R\text{,}\)

\begin{equation*} \begin{array}{rcccl} \displaystyle{ (1+x^r)^a } &=& \displaystyle{ \sum_{n=0}^{\infty} {a \choose n} x^{nr} }&=& \displaystyle{ 1 + {a \choose 1} x^r + {a \choose 2} x^{2r} + {a \choose 3} x^{3r} + \cdots } \\ \end{array} \end{equation*}
