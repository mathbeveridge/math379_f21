Let \(L\) be a lattice path from \((0,0)\) to \((r,s)\text{.}\) Define \(\)f(L) = \{ k \mid \mbox{the } k\mbox{th step is a rightward step} \}.\(\) There are \(r\) such steps, \(f(L)\) is an \(r\)-set chosen from \([r+s]\text{.}\) The inverse of \(f\) is the function \(g: Y \rightarrow X\) which takes an \(r\)-set \(S\) to the lattice path \(L\) whose rightward steps occur at the indices contained in \(S\text{.}\) I leave the details to you.
