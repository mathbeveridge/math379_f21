Item 16.1.a.

Suppose that we have \(n\) prospective jurors, each assigned a number. We want to split the jurors into some number of intervals (the first \(k_1\) jurors, the next \(k_2\) jurors and so on).

  1. Use the composition result above to show that the ordinary generating function \(B(x)\) for this process is

    \begin{equation*} B(x) = 1 + \frac{x}{1-2x}. \end{equation*}

    (Hint: task \({\cal A}\) is "do nothing.")

  2. Find the value of \(b_n\text{.}\)

  3. Give a combinatorial proof of your value in (ii). (Hint: if you haven't yet done so, simplify your answer in (ii) to a single term.)
