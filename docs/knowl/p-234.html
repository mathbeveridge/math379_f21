The Stirling number of second kind \(S(n,k)\) is the number of partitions of the set \([n]\) into exactly \(k\) parts. Here are the first rows of the Stirling triangle:

\begin{equation*} \begin{array}{lccccccccccc} n&S(n,k)\\ \hline \\ 0 & & &&&& 1 \\ 1 & &&&& 0 && 1 \\ 2 & &&& 0 && 1 && 1 \\ 3 & && 0 && 1 && 3 && 1 \\ 4 && 0 && 1 && 7 && 6 && 1 \\ \end{array} \end{equation*}
