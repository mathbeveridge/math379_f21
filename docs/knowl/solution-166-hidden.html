We must have \(a_0=0\) (to use the theorem). For \(n\geq 1\text{,}\) we have \(a_n=1\text{,}\) since there is one way to "do nothing." So

\begin{equation*} A(x) = \sum_{n=1}^{\infty} x^n = x \sum_{n=1}^{\infty} x^{n-1} = x \sum_{m=0}^{\infty} x^m = \frac{x}{1-x} \end{equation*}

and

\begin{equation*} B(x) = \frac{1}{1-A(x)} = \frac{1}{1- \frac{x}{1-x}} = \frac{1-x}{1-2x} = 1 + \frac{x}{1-2x} \end{equation*}
