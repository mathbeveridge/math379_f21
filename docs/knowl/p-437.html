Solve the recurrence relation

\begin{equation*} h_n = 5 h_{n-1} - 6h_{n-2} \mbox{ for } n \geq 2 \end{equation*}

where \(h_0=1\) and \(h_1=2\text{.}\)

  1. Let \(H(x)\) be the generating function for this series. Find a simple closed formula for \(H(x)\text{.}\)

  2. Use the Taylor series for the function in part (a) to find a simple closed formula for \(h_n\text{.}\)

  3. Now solve the same recurrence where \(h_0=1\) and \(h_1=-2\text{.}\) You will need to use partial fractions.
