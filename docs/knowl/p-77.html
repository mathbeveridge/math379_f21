\begin{equation*} {n \choose 4} \end{equation*}

If we choose 4 vertices, there is exactly one way to pair them up to get crossing diagonals. If our vertices in clockwise order are \(v_1, v_2, v_3, v_4\) then we must pair \(v_1\) with \(v_3\) and \(v_2\) with \(v_4\text{.}\) So there is a bijection between intersections of diagonals and 4-sets of vertices.
