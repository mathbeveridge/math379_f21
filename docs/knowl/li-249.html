Item 15.1.a.

Fixing a particular integer \(n\text{,}\) we realized that we can find all of the integer partitions of \(n\) by using the generating function

\begin{equation*} Q_n(x) = \prod_{k=1}^{n} \frac{1}{1-x^k}. \end{equation*}
