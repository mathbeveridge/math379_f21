Solution 1.

We have \(n\) copies of \((x_1+x_2+\cdots+x_s)\text{.}\) In order to contribute to the coefficient of \(x_1^{n_1} x_2^{n_2} \cdots x_s^{n_s}\text{,}\) we must choose \(x_i\) exactly \(n_i\) times (out of \(n\) options) for \(1 \leq i \leq s\text{.}\) The number of ways to do this

\begin{equation*} { n \choose n_1, n_2, \ldots, n_s} = \frac{n!}{n_1! \, n_2! \, \cdots, \, n_s !}.\text{.} \end{equation*}

We then have to sum over all the ways to decompose \(n = n_1 + n_2 + \cdots + n_s\text{.}\)
