Prove the multinomial theorem

\begin{equation*} (x_1 + x_2 + \cdots + x_s)^n = \sum_{n_1 + n_2 + \cdots + n_s = n} { n \choose n_1, n_2, \ldots, n_s} x_1^{n_1} x_2^{n_2} \cdots x_s^{n_s} \end{equation*}

where

\begin{equation*} { n \choose n_1, n_2, \ldots, n_s} = \frac{n!}{n_1! \, n_2! \, \cdots, \, n_s !}. \end{equation*}
