Item 17.7.c.

Let \(A(x) = \sum_{n=0}^{\infty} a_n x^n\text{,}\) the ordinary generating function for coin pyramids. Starting with the recursion above, find a simple closed formula for \(A(x)\text{.}\) Hint: you should start by explaining why

\begin{equation*} \sum_{j=1}^{n-1} (n-j) a_j = \sum_{j=0}^{n} (n-j) a_j \end{equation*}

for this particular sequence.
